External Interfaces

Llama / Tasker / Locale

Apps like the ones mentioned above allow to execute actions based on external events. Those events may well be the connection to a WLAN, the entering of some area. Those events are pretty useful if combined with an FHEM instance. This is why andFHEM provides an external interface to those apps.

UI based

Since version 2.1.4 andFHEM provides a UI for sending commands to FHEM. Just look for an andFHEM plugin in your favourite location aware switching app.

You can also trigger actions based on device states. However, this works only in Locale and Tasker.

Manual Switching

In Llama, you have the option to send own android intents as event. Here are the details you need to enter to send actual commands to FHEM:
PresetOwn Android Intent
Intent-Send-ModeStart Service (Activity or Broadcast will NOT work!)
Package Nameli.klass.fhem
Class Nameli.klass.fhem.service.intent.SendCommandIntentService
Eventli.klass.fhem.constants.Actions.EXECUTE_COMMAND
Under 'Extras' specify a new key value pair containing Key: COMMAND
Datatype: string
Value: <ENTER YOUR COMMAND HERE> (i.e. set abc on)
